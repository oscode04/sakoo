## 🔧 Prasyarat

Pastikan telah menginstal:

- Node.js dan npm

---

## 🚀 Instalasi dan Menjalankan Aplikasi

```bash
# Clone repository
git clone https://github.com/firza-nrk/Sakoo.git
cd Sakoo

# Pindah ke branch front-end
git checkout Front-end
git pull origin Front-end

# Instalasi dependensi
npm install

# Jalankan aplikasi
npm run dev
```
